=====================================================
                 CRUD EM LARAVEL
                 Projeto Avaliativo
=====================================================

Este projeto foi desenvolvido como parte de uma prova
prática da faculdade, com o objetivo de aplicar os
conceitos essenciais do framework Laravel.

O sistema implementa um CRUD básico para fins de
aprendizado, permitindo compreender o fluxo interno
do framework, organização MVC e manipulação de dados.

-----------------------------------------------------
Tecnologias Utilizadas
-----------------------------------------------------
- PHP
- Laravel
- MySQL
- Blade Templates
- Bootstrap (se usado)
- Composer

-----------------------------------------------------
Objetivo do Projeto
-----------------------------------------------------
O intuito deste trabalho é reforçar:

• Estrutura MVC do Laravel
• Uso de migrations e seeders
• Criação de rotas e controllers
• Operações CRUD completas
• Manipulação de views com Blade
• Boas práticas iniciais no framework
