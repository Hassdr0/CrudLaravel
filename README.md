CRUD em Laravel — Projeto Avaliativo

Este projeto foi desenvolvido como parte de uma prova prática da faculdade, com o objetivo de aplicar conceitos fundamentais do framework Laravel, incluindo estrutura MVC, rotas, migrations, controllers e operações CRUD.

O sistema implementa um CRUD simples, servindo como exercício para fixar a lógica do framework e entender melhor o fluxo de uma aplicação Laravel.

Tecnologias Utilizadas

PHP

Laravel

MySQL

Blade Templates

Bootstrap (se aplicado)

Objetivo do Projeto

O foco principal deste projeto é praticar:

Estrutura de pastas e fluxo interno do Laravel

Criação de migrations e manipulação de banco de dados

Implementação de rotas e controllers

Operações CRUD completas

Renderização de views com Blade
